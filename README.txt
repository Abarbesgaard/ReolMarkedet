Git workflow:
Jeg har ikke Reolmarkedet på min computer:

  git clone https://github.com/Abarbesgaard/ReolMarkedet.git

Jeg skal oprette min egen branch på min computer:

  git branch NavnetPåDinBranch 
  git checkout NavnetPåDinBranch

Jeg har lavet ændringer og de skal på github:

  git push -u origin NavnetPåDinBranch
